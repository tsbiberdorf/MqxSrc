ColdFire® MCF5225x Cryptographic Acceleration Unit (CAU)

Cryptographic Acceleration Unit (CAU)

The cryptographic acceleration unit (CAU) is a ColdFire coprocessor implementing a set of specialized operations in hardware to increase the throughput of software-based encryption and message digest functions.

The CAU provides support for acceleration of the DES, 3DES, AES, MD5, and SHA-1 algorithms. This selection of algorithms provides excellent support for network security standards (SSL, IPsec). Additionally, using the CAU efficiently allows you to implement any higher level functions or modes of operation (HMAC, CBC, etc.) based on the supported algorithm.
